# Snake Game - Python

Simple Snake game built with Python.

The player controls a snake that moves around the screen, eats food, and grows longer. The game ends if the snake hits the wall or collides with itself.

This project uses the **pygame** library to create the game window, handle keyboard input, and render graphics.

---

## Features

* Classic Snake gameplay
* Keyboard controls using arrow keys
* Random food generation
* Snake grows when it eats food
* Game over when hitting a wall
* Game over when the snake collides with itself

---

## Technologies Used

* Python
* pygame
* random

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/snake-game-python.git
cd snake-game-python
```

2. Install the required library

```bash
pip install pygame
```

---

## Usage

Run the game:

```bash
python snake.py
```

Controls:

* **Arrow Up** → Move up
* **Arrow Down** → Move down
* **Arrow Left** → Move left
* **Arrow Right** → Move right

The goal is to eat the red food squares while avoiding collisions with the walls or the snake's own body.
